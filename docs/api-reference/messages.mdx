---
title: Ingest CostMessage
api: POST /api/v1/messages
description: Universal ingest — single envelope or batch (max 100).
---

Requires permission `log_usage`. Auto-upserts the project by slug. Idempotent when `idempotency_key` is set.

<ParamField body="project" type="string" required>
  Project slug (auto-created if missing)
</ParamField>

<ParamField body="source" type="string">
  `api` \| `mcp` \| `manual` \| `import` (default `api`)
</ParamField>

<ParamField body="idempotency_key" type="string">
  Dedup key within the workspace
</ParamField>

<ParamField body="timestamp" type="string">
  ISO datetime for the event
</ParamField>

<ParamField body="message" type="object" required>
  Discriminated CostMessage — see [CostMessage](/concepts/cost-message)
</ParamField>

You may also send a **JSON array** of envelopes (1–100).

## Responses

**201 (single)**

<ResponseField name="id" type="string" required>
  Ledger row UUID
</ResponseField>

<ResponseField name="amount_usd" type="number" required>
  Resolved USD amount
</ResponseField>

<ResponseField name="project_id" type="string" required>
  Project UUID
</ResponseField>

<ResponseField name="created_at" type="string" required>
  ISO timestamp
</ResponseField>

**201 (batch):** `{ "messages": [ ...same objects... ] }`

| Status | Meaning |
|--------|---------|
| `400` | Invalid JSON |
| `401` / `403` | Auth / permission |
| `500` | Validation or persist failure |

<RequestExample>
```bash
curl -X POST https://api.costmcp.com/api/v1/messages \
  -H "Authorization: Bearer $COSTMCP_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "project": "slideshow-studio",
    "source": "api",
    "idempotency_key": "run-abc-1",
    "message": {
      "type": "usage",
      "provider": "openai",
      "model": "gpt-image-2",
      "unit_type": "image",
      "quantity": 1,
      "estimated_cost": 0.04,
      "feature": "slide_generation"
    }
  }'
```

```bash Expense
curl -X POST https://api.costmcp.com/api/v1/messages \

    "message": {

```

```bash Batch
curl -X POST https://api.costmcp.com/api/v1/messages \
  -H "Authorization: Bearer $COSTMCP_API_KEY" \
  -H "Content-Type: application/json" \

    {

        "estimated_cost": 0.002
      }
    }
  ]'
```
</RequestExample>
